## 2. Политика логирования

Система сохраняет:

* события обработки ресурсов
* ошибки
* метрики производительности

Не сохраняются:

* персональные данные пользователей
* полный текст приватных сообщений


## 3. Работа с персональными данными

Если входной контент содержит персональные данные:

* система пытается удалить или анонимизировать такие данные
* оригинальный текст не используется в ответах

## 4. Защита от prompt injection

При обработке контента LLM не получает прямых инструкций из пользовательского текста.

Используются:

* системные промпты
* фильтрация подозрительных инструкций

## 5. Подтверждение действий

Операции, изменяющие базу знаний (например удаление ресурсов), требуют подтверждения пользователя.
